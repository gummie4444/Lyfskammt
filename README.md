Lyfskammt
=========

###To start:###

Make sure you have `node`and `npm`installed:

Clone the repo:
`git clone https://github.com/gummie4444/Lyfskammt.git`

Run this to install dependencies:
`npm install`


To run the website write:
`node server.js`

Check the site out at:
`http://localhost:8080` 

To debug js code write:
`gulp jshint`
